Soufflet 🪗 — Hilfe

Kurzanleitung für den Einstieg. Alles läuft lokal im Browser — nichts wird hochgeladen.

1Partitur laden

Oben unter „Partitur" eine Datei wählen — .mxl, .musicxml oder .xml. Beim ersten Laden klappt das Einstellungs-Panel automatisch zu, damit mehr Platz für die Noten bleibt. Über „▸ Einstellungen" lässt es sich jederzeit wieder öffnen.

2Knopflayout wählen

„Rechte Hand" und „Linke Hand / Bass" bestimmen, welche der 26 mitgelieferten Knopfbelegungen genutzt wird. Passt keine davon, lässt sich über „Eigenes .keyboard laden" eine eigene Datei einbinden.

„Stimme (Diskant)" erscheint nur, wenn ein Stück mehrere Melodiestimmen im selben System enthält (z. B. eine gehaltene Begleitstimme neben der eigentlichen Melodie). Bei den meisten Stücken bleibt dieses Feld unsichtbar.

3Die Tablatur lesen

Über jeder Note erscheinen ein oder mehrere spielbare Knöpfe (D.E.S.-Notation). Eine unterstrichene Zahl bedeutet Zug, eine nicht unterstrichene Zahl bedeutet Druck. Antippen wählt eine Variante aus — die hervorgehobene Zahl ist die getroffene Wahl, sie wird gespeichert und bleibt beim nächsten Besuch erhalten.

4Akkordhinweise verstehen

Soufflet zeigt drei unterschiedliche Sicherheitsstufen:

  • Echtes Akkordsymbol (z. B. „Am" über den Noten) — aus der Partitur selbst, bestimmt die Balgrichtung direkt.
  • (C) — eine kleine, kursive Anmerkung in Klammern: Soufflet hat die Akkordwurzel selbst aus gestapelten Bassnoten abgeleitet, weil die Partitur kein eigenes Symbol enthält. Deutlich von echten Symbolen unterschieden, damit nichts verwechselt wird.
  • Hinweis in der Statuszeile („keine Akkordsymbole erkannt …") — wenn gar keine Akkordinformation auffindbar ist. Die Balgrichtung folgt dann nur der „Spielweise"-Einstellung.
Diese Stufen sind bewusst getrennt: Soufflet rät nie heimlich — was nicht sicher ist, wird auch so benannt.

5Spielweise & Balgrichtung

„Spielweise" legt fest, ob ein Akkord die Richtung bestimmt (kreuzweise) oder ob einreihig auf der G- bzw. C-Reihe gespielt wird. „SOL-Akkord" und „FA-Akkord" stellen für diese beiden Akkorde gesondert ein, ob beide Richtungen, nur Zug oder nur Druck erlaubt sind.

6Abspielen

„▶ Abspielen" startet die Wiedergabe mit einem eigenen Akkordeon-Klang. Ein grüner Balken läuft über die aktuell klingende Note, die Seite folgt automatisch (Auto-Scroll). „⏸ Pause" hält genau an der aktuellen Stelle, ein erneuter Klick setzt dort fort. „■ Stopp" setzt auf den Anfang zurück.

Läuft die Wiedergabe weiter unten auf der Seite, erscheint unten rechts eine kleine, fest sichtbare Mini-Leiste mit Play/Pause und Stopp — so bleiben die Bedienelemente immer erreichbar, auch wenn die oberen Buttons durch das Auto-Scroll außer Sicht geraten sind.

7Teilen, QR-Code, Drucken

„Teilen" öffnet das native Teilen-Menü des Geräts (oder kopiert den Link, falls nicht verfügbar). „QR-Code" zeigt einen scanbaren Code zur Live-Adresse — praktisch, um die App in einer Session schnell weiterzugeben. „Drucken" öffnet den Druckdialog mit einer aufgeräumten Ansicht (nur Noten + Tablatur, keine Bedienelemente).

8Sprache wechseln

Die vier Flaggen oben rechts (🇩🇪 🇫🇷 🇮🇹 🇬🇧) schalten die Oberfläche um. Die Wahl wird gespeichert.

9Alles zurücksetzen

Setzt Layout, Optionen, Partitur und alle getroffenen Auswahlen auf den Ausgangszustand zurück — für einen sauberen Neustart.
